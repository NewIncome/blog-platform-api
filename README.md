# Blog Platform API
Java API project to connect to blog react UI

## Main Dependencies
1. Spring Web:
For a Rest API
2. Spring Data JPA:
To transact with the DB
3. PostgreSQL Driver:
To connect to the DB
4. H2 Database:
For in-memory tests
5. Lombok:
To make development quicker
6. Validation:
For Annotation-Driven Validations
